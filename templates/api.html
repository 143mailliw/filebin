Upload file

Specific filename and tag

The following command will upload /var/tmp/some file to the tag exampletag. The target filename will be set to myfile. 

                
$ curl -H "tag: exampletag" -H "filename: myfile" --data-binary "@/var/tmp/some file" {{ .Ctx.Baseurl }}
Auto-generated tag and target filename

The following command will upload /var/tmp/some file, and the tag will be automatically generated. The target filename is not specified, so it will become the SHA256 checksum of the content.

                
$ curl --data-binary "@/var/tmp/some file" {{ .Ctx.Baseurl }}

Show tag meta data

The following command will show the metadata of the tag exampletag.

                
$ curl -H "content-type: application/json" {{ .Ctx.Baseurl }}/exampletag

Download file

The following command will download somefile from the tag exampletag. The file will be stored as myfile.

                
$ curl -o myfile {{ .Ctx.Baseurl }}/exampletag/somefile

Download tag

The following command will download all files in the tag exampletag. The file will be stored as the zip archive mytag.zip.

                
$ curl -o mytag.zip {{ .Ctx.Baseurl }}/archive/exampletag
$ unzip mytag.zip

Delete file

The following command will delete somefile from the tag exampletag. The file will no longer be available at {{ .Ctx.Baseurl }}/exampletag/somefile.

                
$ curl -X DELETE {{ .Ctx.Baseurl }}/exampletag/somefile

Delete tag

The following command will delete the tag exampletag, including all the files in it. The tag will no longer be available at {{ .Ctx.Baseurl }}/exampletag.

                
$ curl -X DELETE {{ .Ctx.Baseurl }}/exampletag
